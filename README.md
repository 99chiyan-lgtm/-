# 失智症照顧與拒絕管理實務挑戰賽

基於實證研究的互動式教學網頁，模擬手機介面，訓練居服員面對失智長者拒絕行為時的溝通技巧。

## 專案結構

```
├── index.html          # 主頁面
├── css/
│   └── style.css       # 樣式表
├── js/
│   └── script.js       # 遊戲邏輯與題庫
├── assets/             # 靜態資源
├── package.json
└── README.md
```

## 使用方式

直接以瀏覽器開啟 `index.html` 即可遊玩。

或執行開發伺服器：

```bash
npm start
```

## 題庫內容

3 大日常照顧難關，涵蓋中度至重度失智情境，每題附實證研究回饋。
