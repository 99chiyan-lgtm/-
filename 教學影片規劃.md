# 失智症拒絕管理實務 — 教學前導影片規劃

## 影片基本資訊

| 項目 | 內容 |
|------|------|
| 片名 | **失智症照顧：當長者說「不要」時** |
| 時長 | 約 3 分鐘 |
| 風格 | 知識型動畫 + 實景情境融合 |
| 目標對象 | 居服員、家庭照顧者 |
| 核心訊息 | 拒絕是溝通，不是症狀 — 用實證策略化解照顧困境 |

## 分鏡表 & 旁白稿

### 第 1 幕：開場 — 問題意識（0:00–0:35）

| 鏡頭 | 畫面 | 旁白 | 字幕 |
|------|------|------|------|
| 1.1 | 深色背景浮現「失智症照顧」標題，搭配輕柔鋼琴 | （無旁白，純音樂開場） | 失智症照顧 |
| 1.2 | 動畫：一位居服員走向長者，長者舉手搖頭說「不要」 | 在失智症居家照顧中，將近七成的家屬與居服員，都曾遇過長者拒絕照顧的情況。 | 近 7 成照顧者曾遇拒絕行為 |
| 1.3 | 長者表情從抗拒轉為痛苦，居服員手足無措 | 不吃藥、不洗澡、不出門——這些「我不要」的背後，其實是長者在表達什麼？ | 拒絕背後，是未被滿足的需求 |
| 1.4 | 畫面轉亮，浮現問號轉換為燈泡圖示 | 今天，我們要從國際實證研究與對話分析中，找到破解拒絕的關鍵策略。 | 破解拒絕的關鍵策略 |

### 第 2 幕：核心策略一 — 順延時間法（0:35–1:05）

| 鏡頭 | 畫面 | 旁白 | 字幕 |
|------|------|------|------|
| 2.1 | 動畫：居服員拿營養飲品，長者搖頭 | （情境還原）奶奶，吃藥了——我不要！ | 「我不要！」 |
| 2.2 | 居服員微笑放下杯子，畫面出現「順延」二字特效 | 當長者明確拒絕時，不要硬碰硬。接受她的否定，同時保留稍後再試的機會。 | 先接受，再延後 |
| 2.3 | 時間快轉，居服員過一會兒再次嘗試，長者配合 | 實證研究指出，這種「順延時間法」能有效保留任務的可完成性，避免陷入權力對抗。 | 順延時間法 |
| 2.4 | 對比動畫：另一場景居服員用疊字催促，長者更加抗拒 | 相反地，使用疊字哄騙的「長者腔」或權威式催促，反而會顯著激怒長者，引發更強烈的抗拒。 | ⚠ 避免：長者腔、權威命令 |

### 第 3 幕：核心策略二 — 降級提案（1:05–1:35）

| 鏡頭 | 畫面 | 旁白 | 字幕 |
|------|------|------|------|
| 3.1 | 動畫：長者坐沙發看電視，居服員提議去公園 | 爺爺，我們去散步好嗎？我今天哪都不去！ | 抗拒高強度活動 |
| 3.2 | 居服員改用輕鬆語氣提議在門口站一下 | 研究發現，將高強度的任務「降級」為低門檻版本，能大幅降低長者的參與阻力。 | 任務降級提案 |
| 3.3 | 長者在門口站著吹風，露出一絲微笑 | 例如：散步 → 門口站一下；洗澡 → 床邊溫熱毛巾拭浴。把目標變小，成功機會就變大。 | 降低門檻，維持活動 |

---

### 第 4 幕：核心策略三 — 環境改良 + 音樂介入（1:35–2:10）

| 鏡頭 | 畫面 | 旁白 | 字幕 |
|------|------|------|------|
| 4.1 | 動畫：長者在浴室門口驚恐退縮 | 面對重度失智長者對洗澡的恐懼，傳統做法是強行推進浴室。（打叉） | ❌ 強行沐浴 |
| 4.2 | 畫面切換：居服員改用溫熱毛巾在床邊替長者拭浴 | 國際知名的「不再為洗澡作戰」研究證實，改用床邊毛巾拭浴，能降低高達 50%–60% 的沐浴衝突。 | ✅ 床邊毛巾拭浴 |
| 4.3 | 畫面加入音符環繞，長者表情放鬆 | 同時，在照顧過程中播放長者喜愛的音樂，是最有實證支持的介入方式，能顯著減少攻擊行為。 | 🎵 個別化音樂介入 |
| 4.4 | 三種策略並列畫面，打勾符號 | 降低 53% 到 60% 的抗拒行為（實證數據來源：Backhouse 2021, Sloane 2004） | 實證成效：降低 50–60% |

### 第 5 幕：溝通的核心 — 拒絕是對話的一部分（2:10–2:40）

| 鏡頭 | 畫面 | 旁白 | 字幕 |
|------|------|------|------|
| 5.1 | 對話氣泡動畫，展現「提出→拒絕→調整→合作」的循環 | 拒絕不是症狀，而是對話的一部分。台灣本土的對話分析研究發現，照顧者如何接住這個「不要」，決定了接下來互動的成敗。 | 拒絕 ≠ 症狀，而是互動的轉折點 |
| 5.2 | 動畫展示：沉默也是一種拒絕信號 | 有時長者的沉默，也是一種拒絕。專業的照顧者會觀察這些信號，並調整下一步。 | 覺察沉默信號 |
| 5.3 | 對比兩種溝通風格：控制 vs. 合作 | 減少控制性語言，多用開放式邀請——照顧是一場雙人舞，而不是單方面的指令。 | 合作代替控制 |

## 素材建議

### 動畫風格

| 類型 | 建議 | 原因 |
|------|------|------|
| **主要風格** | 2D 動畫（Motion Graphics）+ 簡約向量人物 | 成本適中，適合知識傳遞，人物表情可清楚傳達情緒 |
| **色調** | 莫蘭迪粉色系（與網頁一致：#F5EBE6, #E8D7D0, #C5A89E） | 品牌一致性，柔和溫暖減少醫療感 |
| **人物設計** | 扁平風格（Flat Design），圓潤線條 | 親和力高，不失專業 |
| **資訊呈現** | 圖標 + 關鍵字浮現（Icon Animation） | 強化記憶點，適合手機觀看 |
| **過場** | 柔和淡出/滑入，搭配輕微縮放 | 避免眩暈，符合高齡主題 |

### 可套用的剪映素材關鍵字

在剪映「素材庫」中搜尋以下關鍵字即可找到合適素材：

- 動畫背景：`柔和幾何`、`莫蘭迪背景`、`簡約粒子`
- 人物素材：`扁平人物`、`醫護角色`、`老年人向量`
- 圖標元素：`醫療圖標`、`對白氣泡`、`打勾/打叉動畫`
- 過場：`輕柔過場`、`模糊轉場`

### 配樂建議

| 段落 | 音樂風格 | 剪映可搜尋的關鍵字 |
|------|----------|-------------------|
| 開場 0:00–0:10 | 鋼琴獨奏，緩慢溫暖 | `溫馨鋼琴`、`紀錄片鋼琴` |
| 問題陳述 0:10–0:35 | 弦樂鋪底，略帶懸念 | `敘事弦樂`、`情緒鋪墊` |
| 策略解說 0:35–2:10 | 輕快木琴 + 吉他，明亮向上 | `知識科普`、`輕快學習` |
| 尾聲 2:10–3:00 | 鋼琴 + 大提琴，溫暖收束 | `暖心結尾`、`感動時刻` |

> 所有配樂請使用「無版權音樂」或剪映內建曲庫，避免版權問題。

## 用 Suno AI 產製背景音樂

### Suno 提示詞（可直接複製貼上）

#### 版本一：溫馨鋼琴（開場與結尾用）
```
Style: Gentle piano, emotional soundtrack, slow tempo, warm, cinematic, minimal
Lyrics:
(Instrumental - no lyrics)
```
> 在 Suno 的 Style 欄貼上，Lyrics 打 `(Instrumental)` 即可生成純音樂。

#### 版本二：輕快學習風（策略解說段落用）
```
Style: Light acoustic guitar, soft percussion, uplifting, educational background, warm, hopeful
Lyrics:
(Instrumental - no lyrics)
```

#### 版本三：完整混音（整支影片用同一首）
```
Style: Cinematic ambient, warm piano, soft strings, emotional documentary, Taiwanese soundtrack style, slow build
Lyrics:
(Instrumental - no lyrics)
```

### 使用步驟
1. 前往 [suno.ai](https://suno.ai) 或打開 Suno App
2. 點選「Create」
3. 貼上上述 Style 提示詞
4. 確保 Mode 設為 Instrumental（純音樂）
5. 生成後試聽，挑選最適合的版本
6. 下載音檔（MP3），放入 `assets/` 資料夾備用
7. 若要在網頁版播放，參考以下方式嵌入：

```html
<!-- 在教學影片.html 中加入背景音樂 -->
<audio id="bgm" loop>
  <source src="assets/background-music.mp3" type="audio/mpeg">
</audio>
<script>
  const bgm = document.getElementById('bgm');
  // 使用者點擊播放時才開始 (避免自動播放被瀏覽器阻擋)
  document.addEventListener('click', () => { if(bgm.paused) bgm.play(); }, { once: true });
</script>
```

## 剪映技術方案（製作流程）

### 第一階段：素材準備（約 1–2 天）

```
□ 撰寫/確認腳本（本文件已完成）
□ 錄製旁白（剪映「錄音」功能 or 手機錄音匯入）
   - 建議：使用 iPhone 語音備忘錄 or 剪映內建錄音
   - 環境：安靜房間，距離麥克風 15–20cm
   - 口吻：沉穩、溫暖、略慢（每分鐘約 150–180 字）
□ 蒐集素材：
   - 剪映素材庫搜尋上述關鍵字
   - 如需自製動畫：使用 Canva 或 PowerPoint 產出 PNG 序列
   - 現成免費素材網站：Pixabay Videos, Mixkit, Coverr
```

### 第二階段：剪輯流程（剪映操作步驟）

```
步驟 1：建立專案
  打開剪映 → 「開始創作」→ 比例選 9:16（手機直立）
  → 解析度 1080p

步驟 2：匯入素材
  點「導入」→ 選取所有圖片/影片/音檔素材

步驟 3：套用旁白
  將錄好的旁白音檔拖入音軌 → 對照分鏡逐段剪開（分割工具）
  → 調整每段旁白的長度

步驟 4：對位畫面
  將對應的動畫/圖片素材拖入畫軌 → 根據旁白節奏調整長度
  → 關鍵處加上字幕（剪映「文本」→「新建文本」）

步驟 5：加入字幕
  使用「智能字幕」功能 → 選擇「識別字幕（僅字幕）」→ 自動轉寫中文
  → 調整字型：建議「思源黑體 Bold」或「台北黑體」
  → 字體顏色：#5A4B45，背景半透明黑底

步驟 6：特效與動畫
  - 入場動畫：選「見入」「輕柔放大」或「模糊出現」（每段素材皆設定）
  - 轉場：選「淡化」或「模糊」（0.5 秒）
  - 關鍵字強調：使用「動畫」→「循環動畫」→「搖擺」或「呼吸」

步驟 7：背景音樂
  將選好的配樂拖入音樂軌 → 調整音量（約 -20dB 至 -25dB）
  → 確保旁白清晰（旁白 -6dB ~ -3dB，音樂約 -20dB）

步驟 8：調色
  使用「濾鏡」→「溫暖」類別 → 推薦「奶油」「日系」濾鏡（強度 30-50%）
  → 增加些微「光暈」效果

步驟 9：導出
  → 解析度 1080p，幀率 30fps
  → 格式 MP4
  → 碼率「建議高」
```

### 第三階段：檢核清單

```
□ 旁白與畫面同步
□ 字幕無錯字
□ 音樂音量不蓋過旁白
□ 每段畫面停留時間足夠閱讀字幕
□ 整體時長控制在 2:45–3:15
□ 結尾加入互動挑戰賽引導
□ 匯出後在不同手機上測試播放
```
